# Todo List App

A simple and interactive React-based Todo List application that helps users keep track of their tasks.

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Responsive and user-friendly design

## Technologies Used

- React
- CSS (for styling)
- Local Storage (optional for saving tasks)
